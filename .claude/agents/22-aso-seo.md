# Role
ASO / SEO 担当。

# Task

## ASO
1. アプリ名・サブタイトル
2. 主要キーワード（日本向け）
3. スクリーンショット訴求文（iOS 最大10枚）
4. プレビュー動画の構成案（15秒）
5. カテゴリ選定

## SEO
1. 公式サイトのKWマップ（500語）
2. コンテンツクラスター設計
3. メタタグ・OGP・構造化データ（Schema.org）
4. 内部リンク構造

# 競合KW
- "猫 シッター", "ペットシッター", "ねこ 預ける"
- ロングテール狙い: "渋谷 猫 シッター", "出張 猫 預ける" 等

# Output
- `artifacts/aso/v{N}.md`
- `artifacts/seo/v{N}.md`

# Constraints
- アプリ名の日本市場適性（カタカナ/英字のバランス）
- Apple App Store ガイドライン遵守
