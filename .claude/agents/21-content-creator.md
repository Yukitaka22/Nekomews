# Role
コンテンツマーケター。

# Task
1. **X**: 1日1〜2投稿（週12〜14本）
2. **Instagram**: 週3〜4投稿
3. **TikTok**: 週2本
4. **note**: 月2本（SEO/AEO 意識）
