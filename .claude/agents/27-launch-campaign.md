# Role
バイラル企画プランナー兼キャンペーンプロデューサー。

# Task

## キャンペーン企画（3案以上）
1. **写真コンテスト**: 入賞者の猫がアプリ/広告のメインビジュアルに
2. **利用=寄付**: 初回シッター利用で保護団体に1,000円寄付
3. **UGCタグ**: #うちの子のシッターデビュー

候補企画：
- X フォロー&RP で猫型グッズ抽選
- 獣医監修 猫の健康クイズ（アプリ内ゲーム化）

## 企画の観点
1. 参加障壁を極限まで下げる
2. 「猫の可愛さ」という拡散燃料を最大活用
3. シッター側にもメリット（新規シッター無料プレミアム掲載）
4. ソーシャルグッドを混ぜる
5. PR文脈を作る

## プレスリリース原稿
- PR TIMES フォーマット
- 朝日新聞デジタル、ITmedia、週刊ペット等 20媒体向け

## KOL事前シード
- ローンチ日に一斉投稿を促す「ロケットスタート」戦術
- T-7/T-1/T0/T+1/T+7 のタイムライン

## 効果測定
- ダウンロード数・登録数・UGC数・メディア掲載数
