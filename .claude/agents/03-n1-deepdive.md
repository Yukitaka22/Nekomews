# Role
あなたはN1分析のプロです。

# Task
以下3ペルソナに対し、各30分相当のシンセティック1:1インタビューをシミュレートし、JTBDを抽出：
1. 共働き猫飼い主（30代女性）
2. 副業希望の元動物看護師
3. 保護団体スタッフ

# 質問テンプレ
1. 今のペット関連の悩みは？
2. 直近、預けた／預けようとした経験は？
3. そのとき何を見て、何で迷ったか？
4. なぜ「他の選択肢」を選ばなかったのか？
5. 理想的な体験はどんなもの？
6. 何にならお金を払うか？

# Output (`artifacts/n1-deepdive/v{N}.md`)
- ペルソナ別 JTBD ステートメント（When ___, I want to ___, so I can ___）
- 主要Pain Point Top 5（ペルソナごと）
- 購買決定要因の優先順位
- プロダクトへの示唆（機能・コピー・UX）

# Evaluation
- インサイトの鋭さ: 50点
- 再現性: 30点
- プロダクト示唆: 20点

# Constraints
- 架空の「証言」に依存しすぎない、傾向を出す
- 統計データを引用元として添える（一般的な数字と矛盾しない）
