## 3. PASONA フレーム（LP本文構成案）

### Problem（課題提起）
```
「ペットホテルは ストレスが心配」
「親に預けるのも もう申し訳ない」
「友達に頼むのは気が引ける」

そう思いながら、出張や旅行を諦めていませんか？
```

### Affinity（共感）
```
私たちも同じでした。
大切な家族を、どこに預ければ安心なのか。
ケージの中でずっと不安そうにしている子を見て、
「もう出張は無理かも」と思ったこともあります。
```

### Solution（解決）
```
Nekomews は、あなたの家に来てくれる「猫専門のシッター」を
すぐに見つけられるアプリです。

- 近所の登録シッターを料金・評価で比較
- 予約から決済までアプリ内で完結
- お世話の様子は写真付きで毎日報告
- 万一のための保険も自動で付帯
```

### Offer（提供）
```
事前登録者には、初回予約に使える【シッター利用料20%OFFクーポン】をプレゼント🐾
```

### Narrow（絞り込み）
```
※ 対象は東京23区で猫を飼っている方、最初の1,000名様限定です。
```

### Action（行動）
```
メールアドレスを登録して、リリースを先取りする →
```

運営：【有限会社◯◯】

## 7. コピー運用ルール

### ✅ OK
- 「猫専門」「猫に特化」（客観的事実）
- 「プロのシッター」（資格者のみ）
- 具体的数字（例：東京23区、5,000円〜）

### ❌ NG
- 最高級表現（No.1・唯一・絶対）
- 薬機法抵触（治る・効く・改善する）
- 競合社名との直接比較
- 「誰でも簡単に稼げる」系副業訴求
