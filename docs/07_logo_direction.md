## 0. この文書の目的

ロゴデザイナー（外部委託 or AI生成）への **発注仕様** として使える粒度のブリーフ。
3案を並列検討し、最終1案を選ぶ前提。

---

## 1. ロゴに求める要件

### 1.1 必須要件
- [ ] Nekomews のブランド本質（猫／優しさ／コミュニティ）が伝わる
- [ ] アプリアイコンとして **48×48px でも認識可能**
- [ ] モノクロで使用しても成立する
- [ ] 背景色が変わっても成立する（白・黒・アンバー・画像の上）
- [ ] 猫を思わせるが、個別の種（白猫／黒猫／キジトラ）に偏らない
- [ ] 日本人にも海外にも受け入れられる（和モダン可）

### 1.2 望ましい要件
- [ ] アニメーション化できる要素がある（スプラッシュ画面で動かせる）
- [ ] ブランドに応じて拡張可能（サブブランド・コラボ時）
- [ ] 他のペット系ロゴと明確に差別化されている

### 1.3 やってはいけないこと
- [ ] リアルな猫写真の加工
- [ ] 既存の有名ロゴ（Rover / Airbnb / LINE 等）の類似
- [ ] ハートマーク多用（安っぽくなる）
- [ ] 鋭い角・尖った形状（猫の優雅さと不一致）
- [ ] 漫画・アニメ調のキャラクター顔（ターゲット年齢層と不一致）

## 2. ブランドコンセプト（ロゴが表現すべき）

### 2.1 キービジュアルメッセージ
> **「猫と人が、温かくつながる場所」**

### 2.2 ロゴが内包すべき3要素
1. **猫**（鍵のモチーフ）
2. **つながり**（mews = 集合・コミュニティ）
3. **優しさ**（丸み・柔らかさ）

### 2.3 語感の解釈
- **Neko**：日本語の「猫」、柔らかい音
- **Mews**：英語で「猫の鳴き声の複数」「路地・集合住宅」の二重意味
- → 「猫たちが集まって鳴く場所」を視覚化できる構造

## 3. 3つのロゴ方向性

### 🎨 方向性A：**「モダンな肉球 + ワードマーク」**

#### コンセプト
- 肉球の形状を抽象化・幾何学化したシンボル
- "Nekomews" の英字ワードマークと組み合わせ
- 一番オーソドックスで、アプリアイコンにそのまま使える

#### 視覚表現
```
  ●●
 ●　●●        Nekomews
  ●●●
```
- 肉球を 4つの丸＋下の大きめパッドで構成
- 丸は完全な真円ではなく **気持ちふくよかな楕円**
- カラー：Sunrise Amber → Coral Glow のグラデーション

#### ワードマーク
- フォント：**Poppins SemiBold** or **Gilroy Medium**
- 小文字で **nekomews**（大文字より親しみ）
- 「o」の中に小さな肉球を仕込むバリエーションも検討

#### 強み
- 視認性が高くアプリアイコン映え
- 世界観を強く主張しない（海外展開しやすい）
- SNSアイコンに最適

#### 弱み
- 他の猫系ロゴとの差別化が弱め（肉球は定番すぎる）

---

### 🎨 方向性B：**「"o" がつながった和のワードマーク」**

#### コンセプト
- ワードマーク中心、シンボルは使わない（もしくは最小限）
- "nekomews" の綴りの中で **"o" の丸みを猫の顔にする**
- 複数の "o" 的な丸が **繋がって mews（集合）を表現**
- 和のテイスト・余白を大切にした書体

#### 視覚表現
```
nek○mews
    │
    └─ ○ の中にうっすら猫の耳と目
```
- "o" 文字をベースに、上部に2つの小さな三角（耳）
- 口・鼻・ヒゲは極小でほのめかす程度
- 文字同士が **横に流れるようにつながる**（手書きの書道感）

#### カラー
- Sunrise Amber 単色
- または墨色（`#1F2937`）の一色展開

#### 強み
- 独創性高く記憶に残る
- SNSタイムラインで発見されやすい
- 和モダンの雰囲気で日本市場に刺さる
- タイポグラフィで表現するため、多言語展開時のロゴ差し替えが楽

#### 弱み
- 48×48pxで "o" の中の猫が見えづらい可能性
- モノクロ・白背景のみの場合に可読性が落ちる

---

### 🎨 方向性C：**「ひっそり隠れる猫（mews＝街）」**

#### コンセプト
- **"mews" の語源「街・路地・集合住宅」** を図像化
- シルエットの中に **複数の家と、そこから顔を出す猫たち**
- サブストーリー性が強く、ブランドストーリーのベースになる

#### 視覚表現
```
    ▲▲▲
   /家/家/家\
  |  ..  ..  |
   \__猫__/
```
- 三角屋根の家が 3〜5軒並ぶ街並みシルエット
- 窓や屋根の隙間から **小さな猫の耳・尻尾・目** がこっそり覗く
- 発見する楽しさがあるロゴ

#### カラー
- Sunrise Amber（家並み）＋ Ink（猫のシルエット）
- アンバー系のグラデーションで夕暮れの街感

#### 強み
- 世界観が強く、ブランドストーリーに深みが出る
- プロダクトが広がっても（SNS・カレンダー・EC）世界観で統一できる
- 「私の家にも猫が隠れてる」というユーザー参加型に発展可能

#### 弱み
- 小さなサイズ（アプリアイコン）での視認性に課題
- 複雑なデザインは **汎用性（印刷・刺繍・アイコン）で手間**がかかる
- コストが上がりやすい（リファインに時間）

## 5. 私のおすすめ

### 🥇 **方向性 A（肉球 + ワードマーク）を主軸に、方向性 C の要素を段階的に加える**

理由：
1. MVP段階では **シンプルで認識されやすいことが最優先**
2. A はアプリアイコン・SNSアイコン・印刷すべてに耐える
3. 方向性 C のような世界観は **アプリ内部のイラスト・SNS投稿** で表現していけばいい
4. ブランドが成熟したら C の方向性に寄せてリブランド（よくあるパターン）

### 🥈 挑戦するなら **方向性 B（o のワードマーク）**

- 独創性で目立ちたいなら
- 日本市場の和テイスト好きに刺さる
- ただしアプリアイコンだけ別デザイン（方向性A風の肉球）を用意する運用が必要

## 6. 発注の具体仕様

### 6.1 ロゴデザイナーへの発注内容
- **必須納品物**：
  - マスターロゴ（シンボル＋ワードマーク）
  - アプリアイコン用（1024×1024、正方形に収まる形）
  - SNSプロフィール用（正円クロップ対応）
  - モノクロ版
  - 白抜き版（暗い背景用）
  - ファビコン（32×32）
- **納品形式**：AI / SVG / PNG / PDF
- **使用権**：買い切り・改変可・独占

### 6.2 参考レート
| 依頼先 | 相場 | 納期目安 |
|---|---|---|
| ココナラ・ランサーズ | 2万〜5万 | 1〜2週間 |
| 中堅デザイナー（個人） | 10万〜30万 | 2〜4週間 |
| デザイン会社 | 30万〜100万 | 1〜2ヶ月 |
| コンペ形式（99designs等） | 5万〜15万 | 2〜3週間 |
| AI生成＋調整 | 0〜3万 | 1〜3日 |

### 6.3 推奨フロー
1. **AI生成（Midjourney / Ideogram）で3方向性それぞれ5案ずつ生成**
2. 気に入ったものを **中堅デザイナー** に渡してリファイン
3. トータル 10〜20万・2〜3週間で高品質なロゴが得られる

## 7. 意思決定の進め方

### Step 1（今週中）：方向性の確定
- あなたに A / B / C どれに寄せるかを決めてもらう
- または「A主軸 + B or C のスパイス」で進める

### Step 2（次週）：AI生成で各方向性5〜10案
- Midjourney / Ideogram / DALL-E で試作
- 社内（または信頼できる猫好き 3名）にレビュー

### Step 3（3週目）：デザイナーにリファイン
- 1案を選び、プロに整えてもらう
- 必要なバリエーションを揃える

### Step 4（4週目）：ブランドキットの完成
- ロゴシステム、カラー、タイポグラフィを統一したガイドラインに落とす
- Figma でコンポーネントとして管理
